Application Overview

AccessONIX is a tool for creating ONIX XML files for accessible EPUB formats. The application processes your EPUB file and existing ONIX data to generate a complete ONIX 3.0 file with accessibility information.

Process Steps

  1. Upload Files
    • Upload your accessible EPUB file
    • Upload your existing ONIX XML file
    • Enter the EPUB's 13-digit ISBN
  2. Select Output
    • Basic: Basic ONIX generation with accessibility features
    • Enhanced: Complete ONIX generation including sender and pricing information
  3. Enhanced Information (Enhanced Role Only)
    • Sender Name
    • Contact Name
    • Email Address
    • Product Composition
    • Product Form
    • Language Code
    • Pricing Information (CAD, GBP, USD)

File Requirements

  • EPUB file must be properly formatted and contain accessibility features
  • ONIX file must be valid XML
  • Maximum file size: 16MB
  • ISBN must be 13 digits

Output

The application will generate a complete ONIX 3.0 XML file that includes:

  • All accessibility features detected in the EPUB
  • Updated product identifiers
  • Publisher information (if provided)
  • Pricing information (if provided)
