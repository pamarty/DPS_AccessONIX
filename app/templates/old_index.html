To address the need for ONIX which reflects the accessibility features in Accessible EPUB3, DPS provides AccessONIX. By merging data extracted from the EPUB file with data from the print ONIX record for the title, this tool generates a "remediated" ONIX 3.0 record. Use "File Upload" to upload the ePub to be described and the original ONIX record in .xml form. Click "Generate" to create a new ONIX record in the default "Basic" form. The "Enhanced" option allows for the addition of ONIX elements for supply chain use.

Output Options

Choose "Enhanced" to create a record which includes sender, product and pricing information. Choose "Basic" to create a record without these details.

Enhanced Information

The record created here will pass validation against the ONIX standards. For free ONIX, sign up here.

AccessONIX is free for use by any DPS client, where it may be used without charge to create remediated ONIX for backlist. For frontlist conversions supplied by DPS, remediated ONIX is included in the conversion price.
